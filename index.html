هدف از ساخت Bazaar.af

من اورانوس هستم، با علاقه‌مندی به هنر، فرهنگ و طراحی وب. این پروژه را برای تمرین مهارت‌های طراحی سایت با استفاده از HTML و Bootstrap ساختم. اما در کنار یادگیری، هدفم این بود که گوشه‌ای از هنر زنان افغان را نشان دهم؛ زنانی که با دستان توانمندشان، چپلی، گلدوزی و هنرهایی می‌آفرینند که ارزش دیده شدن دارند. این وب‌سایت، ادای احترامی‌ست به تلاش‌های آنان و گامی کوچک برای زنده نگه داشتن فرهنگ اصیل افغانستان.

مهارت‌های که در این پروژه استفاده کردم

HTML

از HTML برای ساختاردهی محتوای سایت استفاده کردم؛ شامل عنوان‌ها، پاراگراف‌ها، تصاویر و دکمه‌ها.

Bootstrap

با استفاده از بوت‌استرپ صفحات را به‌صورت واکنش‌گرا طراحی کردم و از کلاس‌های آماده برای چینش و استایل استفاده نمودم.

طراحی ظاهر سایت

در این پروژه سعی کردم با ترکیب رنگ‌ها، فاصله‌ها و عناصر بصری، سایتی ساده، خوانا و کاربرپسند بسازم.
